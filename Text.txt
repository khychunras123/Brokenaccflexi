## គោលបំណង

ក្នុងមុខងារ **Submit New Report** ត្រង់ **របាយការណ៍ខូច (Step 3)** មានផ្នែក **Search Product**។ ខ្ញុំចង់អភិវឌ្ឍមុខងារនេះឱ្យភ្ជាប់ជាមួយ API ដើម្បីទាញយក និងស្វែងរកទិន្នន័យផលិតផល (Products) ពី Server។

## API Information

### Endpoint

```http
GET https://oder-backend-2.onrender.com/api/products
```

### Headers

```http
X-API-Key: my-secret-key-ChanRas-123
```

### Permission

* Read Only
* ប្រើសម្រាប់ទាញយកទិន្នន័យតែប៉ុណ្ណោះ
* មិនអាចកែប្រែ ឬលុបទិន្នន័យបានទេ

## Functional Requirements

### 1. Load Products from API

* នៅពេលបើក Step 3 ឬ Search Product Section
* System ត្រូវ Call API ដោយស្វ័យប្រវត្តិ
* ទាញយកបញ្ជី Product ទាំងអស់

### 2. Search Functionality

អាចស្វែងរក Product តាម៖

* ProductName
* Barcode
* Tags

ការស្វែងរកត្រូវជា:

* Real-time Search
* Case-insensitive
* Partial Match

ឧទាហរណ៍:

* Search "123" → រកឃើញ Barcode ដែលមាន 123
* Search "ផលិត" → រកឃើញ ProductName ដែលមានពាក្យនេះ
* Search "New" → រកឃើញ Product ដែលមាន Tag = New

---

### 3. Product Display Card

បង្ហាញព័ត៌មានខាងក្រោមសម្រាប់ Product នីមួយៗ៖

* Product Image
* Product Name
* Barcode
* Price
* Cost
* Tags

ប្រសិនបើ ImageURL មិនមាន ឬ Image Load Failed:

* បង្ហាញ Placeholder Image

---

### 4. Product Selection

នៅពេល User ចុចលើ Product:

* រក្សាទុក Product ដែលបានជ្រើស
* បំពេញព័ត៌មានទៅក្នុង Damage Report Form
* បង្ហាញ Selected Product Summary

## API Connection Test Feature

សូមបន្ថែមផ្នែក **API Connection Test** សម្រាប់ពិនិត្យស្ថានភាពការភ្ជាប់ API។

### Test Button

Button:

```text
Test API Connection
```

### Success Result

```text
✅ API Connected Successfully
```

បង្ហាញ:

* Status
* Total Products Retrieved
* Response Time

### Failed Result

```text
❌ API Connection Failed
```

បង្ហាញ:

* Error Message
* HTTP Status Code
* Retry Button

* Prevent Multiple Duplicate API Requests
